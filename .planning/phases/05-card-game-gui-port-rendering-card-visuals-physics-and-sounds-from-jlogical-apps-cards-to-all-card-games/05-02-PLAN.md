8. **Wire sound effects** in the WebSocket message handler for game state updates. The host receives game state via `GAME_STATE` messages. Add sound triggers by detecting state transitions:

   For **Shithead** (in the `renderShithead` function or the message handler that calls it):
   - When `msg.lastAction === 'play'` or a card was just played → `audio.play('place')`
   - When `msg.lastAction === 'draw'` or draw pile count decreased → `audio.play('draw')`
   - When pile was burned (pile count drops to 0 after being > 0) → `audio.play('deck_redraw')`
   - When `msg.phase === 'finished'` (first time) → `audio.play('win')`

   For **Gin Rummy** (in the `renderGinRummy` function or handler):
   - When `msg.lastAction === 'draw'` → `audio.play('draw')`
   - When `msg.lastAction === 'discard'` → `audio.play('place')`
   - When a new hand starts (round number increases) → `audio.play('deck_redraw')`
   - When `msg.phase === 'scoring'` and result includes gin → `audio.play('win')`
   - When `msg.phase === 'finished'` (game over) → `audio.play('win')`

   Implementation approach: Track `lastRenderedState` as an object storing `{pileCount, phase, round}`. Compare current msg against it to detect transitions. Set `lastRenderedState` at end of each render. This avoids playing sound on every re-render.

   ```js
   // Near top of script, outside any function:
   var lastShState = {};
   var lastGrState = {};

   // Inside renderShithead, before building HTML:
   if (msg.phase === 'play' || msg.phase === 'finished') {
     if (lastShState.pileCount != null && msg.pileCount === 0 && lastShState.pileCount > 0) {
       audio.play('deck_redraw');
     } else if (lastShState.drawPileCount != null && msg.drawPileCount < lastShState.drawPileCount) {
       audio.play('draw');
     } else if (lastShState.pileCount != null && msg.pileCount > lastShState.pileCount) {
       audio.play('place');
     }
     if (msg.phase === 'finished' && lastShState.phase !== 'finished') {
       audio.play('win');
     }
   }
   lastShState = { pileCount: msg.pileCount || 0, drawPileCount: msg.drawPileCount, phase: msg.phase };

   // Inside renderGinRummy, before building HTML:
   if (lastGrState.round != null && msg.round > lastGrState.round) {
     audio.play('deck_redraw');
   }
   if (msg.phase === 'scoring' && lastGrState.phase !== 'scoring') {
     audio.play('win');
   }
   if (msg.phase === 'finished' && lastGrState.phase !== 'finished') {
     audio.play('win');
   }
   if (msg.phase === 'play' && lastGrState.phase === 'play') {
     if (msg.stockCount != null && lastGrState.stockCount != null && msg.stockCount < lastGrState.stockCount) {
       audio.play('draw');
     }
     if (msg.discardCount != null && lastGrState.discardCount != null && msg.discardCount > lastGrState.discardCount) {
       audio.play('place');
     }
   }
   lastGrState = { round: msg.round, phase: msg.phase, stockCount: msg.stockCount, discardCount: msg.discardCount };
   ```
