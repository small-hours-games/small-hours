    2. Constants:


    3. `stableId(question)` function — per D-09, research Pattern 2:
       ```js
       function stableId(question) {
         const key = question.question + '\0' + question.correct_answer;
         return 'otdb_' + createHash('sha1').update(key).digest('hex').slice(0, 12);
       }
       ```

    4. `normalizeIds(questions)` — maps questions array, replacing `id` with stableId:
       ```js
       function normalizeIds(questions) {
         return questions.map(q => ({ ...q, id: stableId(q) }));
       }
       ```

    5. `fetchQuestions(categoryId, amount = 10)` — per D-01, D-04, D-05, D-06, D-07:
       - Compute `cacheKey = categoryId ?? 'any'` and `cachePath = join(CACHE_DIR, cacheKey + '.json')`
       - Try `readFile(cachePath, 'utf8')` → `JSON.parse` → return `{ ok: true, questions: cached }`
       - On ENOENT: fall through silently
       - On other read error: `console.warn('[cache] read error:', err.message)` then fall through
       - On cache miss: call `fetchFromApi(categoryId, amount)`
       - If `!result.ok`: return result immediately (no cache write)
       - Normalize IDs: `const normalized = normalizeIds(result.questions)`
       - Write cache: `await mkdir(dirname(cachePath), { recursive: true })` then `await writeFile(cachePath, JSON.stringify(normalized), 'utf8')`
       - Wrap writeFile in try/catch — on failure: `console.warn('[cache] write error:', err.message)` and continue
       - Return `{ ok: true, questions: normalized }`

    6. `clearCache(categoryId)` — per D-14:
       ```js
       export async function clearCache(categoryId) {
         if (categoryId === undefined) {
           await rm(CACHE_DIR, { recursive: true, force: true });
         } else {
           const key = categoryId ?? 'any';
           await unlink(join(CACHE_DIR, `${key}.json`)).catch(() => {});
         }
       }
       ```
